Website Developer

Role

We are looking for an Website(Angular) Developer to join our team who shares our passion to create a seamless product verification platform for consumers. If you love thinking big and delving deep and enjoy envisioning truly elegant solutions, this role is definitely for you.

Experience

Minimum 1 Year

What you will be Doing

  • Create intuitive interfaces with great functionality in a timely manner.
  • Create mobile friendly web pages that work across devices.
  • Create well-written JavaScript, as well as expand the current JavaScript code.
  • Create and maintain web forms that integrate with our CRM.
  • Assist with API integration for dynamic email/web prospect and customer communications.
  • Translate designs and style guides provided by the UI/UX team into functional user interfaces, while ensuring cross-browser compatibility and performance.
  • Create functional pages and banners from design mock-ups.
  • Test the code across platforms and browsers.
  • Document the work and communicate any issues as and when they arise.
  • Assist in troubleshooting any production issues.

What you need to know

  • Prior 1+ years of experience in front-end development in a start-up or a dynamic work environment.
  • Minimum 1 year of experience using Front End frameworks such as Angular 2.0/ Angular JS.
  • Knowledge of web standards and accessibility (W3C and s508).
  • Ability to create production-quality CSS, SASS, and HTML for web applications with support for mobile and tablet devices.
  • Good understanding of web-based application architecture.
  • Designing/coding to overcome browser/platform constraints.
  • Excellent problem-solving skills and knowledge of JavaScript.
  • Able to collaborate effectively with both technical and non-technical colleagues.
  • Deadline oriented and able to multi-task in a fast paced environment.
